# IrisObjectRotation
> Configures rotation for iris
```json
{
    "yAxis": {
        "min": 0,
        "max": 0,
        "interval": 90,
        "enabled": true
    },
    "xAxis": {
        "min": 0,
        "max": 0,
        "interval": 0,
        "enabled": false
    },
    "zAxis": {
        "min": 0,
        "max": 0,
        "interval": 0,
        "enabled": false
    },
    "enabled": true
}
```

## enabled
> If this rotator is enabled or not

## xAxis
> The x axis rotation

## yAxis
> The y axis rotation

## zAxis
> The z axis rotation
